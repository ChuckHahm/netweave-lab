## Question

Does the `employer_desc` field generated by Claude API during node enrichment (Step 2) improve downstream query ranking quality?

## Hypothesis

`employer_desc` adds modest signal (~2–5% NDCG improvement) but will show rank_correlation > 0.90 with the baseline, suggesting marginal benefit relative to API cost.

## Success criterion

If `with_desc` NDCG@10 ≥ `no_desc` NDCG@10 + 0.02 AND rank_correlation < 0.95, employer_desc is justified.

| Variant | Key parameters |
|---------|----------------|
